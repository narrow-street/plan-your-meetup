Meetup Venues and Where To Find Them?¶

By Ruosi Wang 2019/2/7

When it comes to events, location matters. It could be entirely different for very similar meetups, if one is just a block away ("Hey, I'm there!") and the other is one hour away ("Nah, not worth going"). Therefore, it is critical to consider whether a venue location is close to your audience enough. Here, we created visualizations of venue locations, from which you could target the center areas of different meetup topics.

In [4]:
import viz_explorative as vz
vz.plot_venues('Boston', scope=6000, x_move=-1600, y_move=600, legend_location='top_right',
              topics=['social', 'career-business', 'outdoors-adventure', 'tech'], )

Let's first use Boston as an example to explore the venues that host meetups with the topics of social, career-business, outdoors-adventure and tech. These are the top four topics with the most number of meetup events in Boston. In the above plot, each circle represents one venue location. The color of a circle depicts the meetup topic, and the size of it is proportional to the number of hosted meetups.Therefore, the larger the cirle, the greater the number of events that were hosted.

This plot indicates some exciting patterns of venue geographic distributions.

  • Tech venues (purple) centered mostly around MIT and downtown Boston
  • Career-business venues (yellow) overlapped a lot with tech venues (purple)
  • Sites that host social meetups(red) were more dispersed than other topics

Methods¶

The dataset was fetched from the Meetup API utilizing a python API client. In particular, We first identified the Meetup groups(>100 members) in the top 30 US cities with the most number of groups and then gathered the meetup details associated with these groups between 2011 and 2018 July.

We implemented the above explorative analysis with PySpark, please see here for the source code.
